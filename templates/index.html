Upload Guidelines

  • Format: Must be standard .fasta format and pre-aligned (padded with - gaps so all sequences are the same length).
  • Gene Targeting: Partial or complete sequences are acceptable, but they must all target the exact same gene segment (e.g., only Hemagglutinin).
  • Header Rule (Critical): FASTA headers MUST contain the collection date at the end, separated by a pipe | character (e.g., >id|location|host|date or >id|location|host|2015-09-23). Missing dates will cause the Bayesian MCMC engine to fail.
  • Dataset Size: Absolute minimum of 4 sequences (with different dates) required for linear regression. 30 to 1,000+ sequences recommended for optimal mathematical accuracy.

Mathematical Parameters

1. Year

The specific calendar year the viral sequences were sampled, acting as the temporal anchor for all molecular clock and phylogenetic calculations.

2. Sequence Count (n)

The total number of valid FASTA sequences successfully processed in the alignment block for the given year.

3. Selection Pressure (dN/dS Ratio)

Calculated using a memory-safe variation of the Nei-Gojobori (1986) method. Sequences are translated into amino acids, and synonymous/non-synonymous sites are counted and corrected using the Jukes-Cantor model for multiple substitutions.
Formula: dN/dS = Mean(-0.75 * ln(1 - 4/3 pN)) / Mean(-0.75 * ln(1 - 4/3 pS))

4. Codon Adaptation Index (CAI)

Calculated using the Sharp & Li (1987) formula. The reference weight parameters are hardcoded to the official Gallus gallus (Chicken) Codon Usage frequencies, making this highly accurate for Avian Influenza dynamics in primary avian hosts.
Formula: CAI = exp( (1/L) * Σ ln(w_i) ) where w_i is the relative adaptiveness of the codon.

5. Evolutionary Rate

Calculated via IQ-TREE v2 combined with the Least-Squares Dating (LSD2) algorithm. A Maximum-Likelihood phylogenetic tree is built using the best-fit substitution model (e.g., HKY+F+I+G4). This optimal model is determined automatically for each specific sequence alignment by the ModelFinder algorithm utilizing the Bayesian Information Criterion (BIC). LSD2 is then applied to the time-calibrated tree to calculate the evolutionary rate (substitutions per site per year) across the timeline.
Formula: Rate = Least-squares estimate of substitutions/site/year derived from the time-calibrated phylogeny.

6. Effective Reproductive Number (Re - BDSKY)

Calculated using a Bayesian Markov Chain Monte Carlo (MCMC) algorithm via BEAST 2 utilizing the Birth-Death Skyline (BDSKY) model. This estimates the temporal transmission dynamics of the virus, yielding the effective reproductive number (Re), which signifies whether the viral population is expanding (Re > 1), endemic (Re ~ 1), or declining (Re < 1).
Formula: Posterior Mean of Re from BDSKY MCMC.

7. Recombination Rate (CoalRe)

Calculated using Headless BEAST 2 (CoalRe). A Bayesian MCMC algorithm traverses Ancestral Recombination Graphs (ARGs) and applies Coalescent Theory to calculate the true population Recombination Rate (ρ) or reassortment frequency.
Formula: Posterior Mean of ρ from CoalRe MCMC.

8. Genetic Diversity (π)

Calculated using the standard nucleotide diversity formula (average pairwise differences per valid site across the entire alignment block).
Formula: π = Σ (diffs / valid_sites) / Total_Comparisons

9. Genetic Distance (GD)

Calculated as the raw un-normalized pairwise p-distance between all sequences in the cluster. (Mathematically identical base arithmetic to π).
Formula: GD = Σ (diffs / valid_sites) / Total_Comparisons

10. Mutation Burden (MB)

Calculated by comparing every sequence in a given year to the very first sequence (the root) in that year's dataset, measuring the accumulation of raw mutations over time within the cluster, normalized per 1000 base pairs.
Formula: MB = Mean( (Sequence_Mismatches_vs_Root / Valid_Length) * 1000 )

11. GC Content

Calculated by measuring the exact percentage of Guanine (G) and Cytosine (C) nucleotides across all sequences in the alignment block, which is a primary indicator of viral thermal stability.
Formula: GC = (Count_G + Count_C) / Total_Valid_Nucleotides

12. GVI 2.0 Composite Score

A scalar index that integrates all 9 primary genomic parameters into a single weighted score representing overall virulence and adaptive potential.
Formula: GVI(t) = w₁·Evo_Rate + w₂·Re + w₃·π + w₄·MB + w₅·dN/dS + w₆·GD + w₇·CAI + w₈·Recomb + w₉·GC (normalized to a 0.0-1.0 scale).
